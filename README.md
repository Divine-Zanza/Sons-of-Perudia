# ⚔️ Survival Board Game Team Generator

Simple open-source tool to generate a team of 5 units based on rarity.

## Features
- Weighted rarity system
- No duplicates
- Easy to expand with new units

## Rarity Rates
- Common: 50%
- Rare: 25%
- Epic: 15%
- Legendary: 10%

## How to Use
1. Download or clone repo
2. Open `index.html`
3. Click "Generate Team"
